Student Freelancer Agency

Acest proiect prezintă un grup de studenți care își doresc să deschidă o mică afacere de tip freelancing și, totodată, să își îmbunătățească abilitățile practice.



Funcționalități:

-Navigare între pagini

-Design responsive și modern

-Formular de contact

-Tabel pentru prezentarea serviciilor

-Accesibilitate (skip link)



Structura proiectului:

-index.html – pagina principală

-about.html – informații despre agenție

-data.html – servicii oferite

-contact.html – formular de contact

-styles.css – stilizarea site-ului
